# PYTHON CALCULATOR

A simple terminal-based calculator buil-in Python.

----------

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Input validation
- Prevents division by zero
- Continuous usage loop


## How to Run

```bash
python3 calculator.py
```

----------

## Example Usage 

```text
1. Addition
2. Subtraction
3. Multiplication
4. Division

Choose an option: 1

Addition:
First number: 5
Second number: 3

Result = 8

----------
```
